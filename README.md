Analyze and display the musical scale from the input music score image.

### Requirements

- First, your environment requirements `asdf`.

```sh
$ brew install asdf
```

### Usage

```sh
### [Only after clone] Configure your local environment.
$ make init

### Run this program.
$ make run
```
